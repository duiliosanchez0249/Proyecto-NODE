# PROYECTO NODE - Discord Bot

### Descripción
Mi proyecto NODE consiste en crear un bot de Discord que ayude al usuario a perder el tiempo con chistes rancios, juegos como cara o cruz, entre otros. La idea y de este proyecto fué trabajar, comprender y ver el funcionamiento de los comandos de NODE haciendo un proyecto de creatividad libre, junto con las herramientas de aprendizaje como lo son Visual Studio, Git Bash, entre otras. Se pudo llegar al resultado siguiente.

---

## Comandos del Bot
Aquí tienes la lista de lo que puedes hacer:

| Comando | Descripción |
| :--- | :--- |
| `!chiste` | Suelta un chiste rancio para romper el hielo. |
| `!suerte` | Juega un volado (Cara o Cruz). |
| `!ayuda` | Muestra la lista completa de funciones. |
